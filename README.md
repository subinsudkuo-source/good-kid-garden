# สวนภารกิจเด็กดี ป.2

เกมแบบฝึกหัดแสนสนุก เรื่อง **สมาชิกในบ้านและการช่วยเหลืองานบ้าน** สำหรับนักเรียนชั้นประถมศึกษาปีที่ 2

## ภายในเกม

- 6 ด่านทบทวนความรู้
- คำถามแบบเลือกตอบ จริงหรือไม่ จับคู่ และเติมคำ
- เกมใบ้คำงานบ้าน
- ภารกิจเด็กดีประจำสัปดาห์
- ระบบสะสมดาวและสวนที่เติบโตตามความสำเร็จ
- บันทึกความคืบหน้าไว้ในอุปกรณ์ของผู้เล่น
- รองรับคอมพิวเตอร์ แท็บเล็ต และโทรศัพท์
